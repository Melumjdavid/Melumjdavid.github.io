Design Considerations

This website demonstrates several key graphical design elements and principles. First, we used a consistent layout structure across all pages using Bootstrap's responsive grid. Second, the font was changed to 'Roboto' to improve readability and maintain a modern feel. Third, a subtle linear gradient background gives the site depth without distracting from content. Fourth, consistent nav and footer placement aids user navigation and expectations. Fifth, all images are floated for visual balance and wrapped with text. Sixth, we ensured the site is mobile responsive using Bootstrap containers, rows, and columns. Seventh, accessibility was addressed by using semantic HTML and alt text on images. The site passes HTML validation with no errors. Lighthouse scores were high across all categories.
